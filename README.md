# 5hmC2Stage

**5hmC2Stage** is a two-stage framework for cell-group-specific
5-hydroxymethylcytosine (5hmC) prediction from paired single-cell
methylome data.

Stage 1 learns a continuous site-level Propensity Score from
CpG-centered sequence, genomic context and transcription-factor motif
features. Stage 2 integrates this score with cell-group-specific 5mC
features to estimate the probability of a high-5hmC state for each CpG
site within each cell group.

## Data

The paired single-cell 5mC and 5hmC datasets analyzed in the manuscript
were obtained from the Gene Expression Omnibus under accession
**GSE197740**.

Reference genome assemblies:

- Mouse Brain: mm10 / GRCm38
- Human PBMC: hg38 / GRCh38

Processed model-input tables, Stage-1 out-of-fold Propensity Scores,
Stage-2 and matched-baseline prediction outputs, regional comparison
files and motif-analysis outputs are archived on Zenodo:

- Dataset, version 1.0.0: https://doi.org/10.5281/zenodo.21373461

The required reference genomes, gene annotations and motif databases
must be obtained separately.

## Installation

The code was developed with **Python 3.9.1**.

Create and activate a virtual environment:

```bash
python -m venv .venv
```

Windows PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

Linux or macOS:

```bash
source .venv/bin/activate
```

Install the required packages:

```bash
python -m pip install -r requirements.txt
```

The primary models use CatBoost 1.2.8 for Stage 1 and LightGBM 4.6.0
for Stage 2. PyTorch is required for the Deep5hmC-inspired regional
comparator.

## Workflow

The scripts are organized in execution order.

### Data preparation and label construction

```text
P0_data_prepare.py
P1_build_long_table_and_labels.py
P1b_export_continuous_signal.py
```

These scripts prepare cell-group-level 5mC and 5hmC tables, construct
cell-group-specific labels and export continuous signals.

### Feature construction

```text
P2_extract_cpg_sequences.py
P2a_build_sequence_features.py
P2b_build_genomic_context.py
P2c_build_motif_features.py
P3a_build_site_5mc_features.py
P3b_build_promoter_5mc_features.py
```

These scripts generate CpG-centered sequence features, genomic-context
features, motif-derived features, site-level 5mC features and
promoter-level 5mC features.

### Model input construction

```text
P4_build_stage1_dataset.py
P4_build_stage2_dataset.py
```

These scripts construct the final Stage-1 and Stage-2 input tables.

### Stage 1 and out-of-fold Propensity Scores

```text
P5_train_stage1.py
P5_generate_stage1_oof_scores.py
```

Stage 1 uses CatBoost to learn a continuous site-level Propensity Score.
Genomic partitions are defined using non-overlapping 10-kb blocks. The
scores supplied to Stage 2 are generated by five-fold out-of-fold
prediction with genomic blocks as grouping units. Each CpG site is
therefore scored by a model that was not trained on its own genomic
block.

### Stage 2

```text
P5_train_stage2.py
```

Stage 2 combines the out-of-fold Stage-1 Propensity Score with
site-level 5mC, promoter-level 5mC, relative and coupling features, and
cell-group information.

### Matched baselines and regional comparison

```text
P5_train_matched_baselines.py
P6_train_deep5hmc_inspired_comparator.py
P6_evaluate_regional_comparison.py
```

These scripts implement the matched one-stage baseline comparison and
the regional-scale comparison reported in the manuscript.

Run a script with `--help` to inspect its available arguments, for
example:

```bash
python src/P5_train_stage1.py --help
```

## Example

The `examples/` directory contains small subsets of the Brain input
tables. The files demonstrate the expected input format and execution
process; they are not intended to reproduce the manuscript results.

Generate example out-of-fold Stage-1 Propensity Scores with:

```bash
python src/P5_generate_stage1_oof_scores.py --stage1_csv examples/stage1_input_example.csv --score_csv examples/stage1_input_example.csv --out_csv examples/stage1_oof_scores_example.csv --out_dir examples/oof_output --models cat --score_output cat --folds 5 --seed 42 --block_bp 10000
```

The command generates:

```text
examples/stage1_oof_scores_example.csv
examples/oof_output/stage1_oof_metrics.csv
examples/oof_output/stage1_oof_summary.csv
```

Because the example dataset is deliberately small, its AUC and AP values
should not be compared with those reported in the manuscript.

## Results

The `results/` directory contains compact CSV summaries corresponding to
the main manuscript tables:

- `Table 1/`: Stage-2 baseline and full-model performance
- `Table 2/`: cell-group-level performance
- `Table 3/`: matched one-stage baseline comparison
- `Table 4/`: regional comparison after 1-kb aggregation

Complete processed inputs and analysis outputs are available from the
Zenodo dataset listed above.

## Reproducibility notes

- Genomic partitions use non-overlapping 10-kb blocks.
- Stage-1 scores supplied to Stage 2 are generated by five-fold
  genomic-block out-of-fold prediction.
- Repeated evaluation settings and random seeds are recorded in the
  manuscript and archived run configurations.
- Local paths must be replaced with valid paths before execution.
- Reference genomes, gene annotations and motif databases are not
  distributed in this repository.
